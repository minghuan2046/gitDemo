# iostat

## 作用：
I/O statistics缩写，即输入输出统计，可用于查看CPU，磁盘等活动情况。

## 命令参数

    -c:查看CPU使用情况  
    -d:查看磁盘使用情况  
    -k:以KB为单位
    -m:以M为单位
    -x:显示详细信息
## 属性说明

## 实例
### 1.查看某一磁盘状态  
iostat -d nvme0n1

output:
```
Device             tps    kB_read/s    kB_wrtn/s    kB_read    kB_wrtn
nvme0n1           7.14        68.73        84.22    3289206    4030613
```
其中tps表示该设备每秒传输次数，其中多次逻辑IO请求可能被合并为同一个IO请求。
